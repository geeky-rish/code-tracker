☁️ Automatic Mobile & Desktop Sync

Enter a GitHub Personal Access Token (PAT) below. When added, any change on Desktop or Mobile automatically commits to your GitHub repo and keeps all devices in 100% real-time sync!

Stored only inside your browser's private localStorage. Never exposed.
How to get a GitHub Token (30 seconds):
1. Go to GitHub Settings → Developer Settings → Personal Access Tokens
2. Click Generate New Token
3. Grant Repository Permissions: Contents (Read & Write)
4. Copy & paste the token here and on your mobile browser!
