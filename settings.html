How to set up MongoDB Atlas Data API (60 seconds):
1. Open your MongoDB Atlas Dashboard.
2. Go to App Services (top tab) → Create a new App (name it `placement-quest-api`).
3. Go to HTTPS Endpoints or Data API → Click **Enable Data API**.
4. Generate an API Key and copy the **App ID** (found in App Services UI). Paste them here and on your mobile device!
